# AI Tooling

This project was developed with assistance from Codex.

Codex was used to:

- Interpret the project requirements and rubric
- Create the architecture guide
- Plan the MVC interaction flow
- Scaffold the Docker-based project structure
- Generate the initial React frontend, Flask backend, PostgreSQL schema, and Nginx ingress configuration

The AI assistance worked well for creating a complete starting point quickly. Human review is still needed for final styling choices, testing, demo preparation, and any deployment-specific configuration.
